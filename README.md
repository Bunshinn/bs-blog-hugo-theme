# bs-blog-hugo-theme

## Mermaid支持  
使用hugo的shortcodes语法
```
{{< mermaid >}}
graph LR
A-->B
{{< /mermaid >}}
```

## 使用方法  
hugo建站后,进入themes文件下, git下载到命名文件来`bs-blog`下:  
`git https://github.com/Bunshinn/bs-blog-hugo-theme bs-blog`  
然后在config.toml文件中添加一行
```
theme = "bs-blog"
```
